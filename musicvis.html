Music Visualizer Demo (WIP)

Personal project created in Processing that uses live computer audio



CONCEPT

Using the Processing Sound library, audio is streamed into Processing and both the amplitude and the FFT of the sound are used as a basis for the generated graphics.

Code for this specific iteration can be found on GitHub.
